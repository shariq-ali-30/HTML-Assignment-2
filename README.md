# Coffee House Website

This is my second HTML assignment created as part of my Web and App Development learning journey at SMIT.

## About The Project

Coffee House is a simple HTML-based website designed for a coffee shop. 
This project focuses on practicing HTML structure, text formatting, lists, quotations, links, and basic webpage organization.

## Technologies Used

- HTML5

## Features

- Coffee shop introduction section
- Menu list using HTML lists
- Special offer section
- Text formatting using:
  - Bold text
  - Italic text
  - Quote tags
  - Deleted text
- Contact information section
- Email link using mailto

## What I Learned

Through this assignment, I learned:

- How to structure a webpage using HTML
- How to use lists for displaying menu items
- How to create clickable email links
- How to use different HTML formatting tags
- How to organize content into sections
